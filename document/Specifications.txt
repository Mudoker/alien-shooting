6. Required commands:
	> help: show all commands (> help)

		-t: change text color
		-b: change bg color

	> ref -os -c -theme: change theme of app
		-consolas:
		-nebula:

	> showinfo:
		-default: show all (Board revison & board Mac address)
		-verbose: show details

	> ref -sel -b: slect a board (show a list of board connected)
		
	> ref -uart<uart_number(1,2,3)>: Configure uart (should check if selected a board)
		-c -baud <bits_per_second>: set baud rate
		-c -dbits <5/6/7/8>: set data bits (5/6/7/8)
		-c -sbits <1/2>: set stop bits (0/1)
		-c -par <none/even/odd>: set parity (non/even/odd)
		-c -handshake <CTS/RTS>: handshaking control (CTS/RTS)

7. Extra commands:
	> ref -os -r: restart os

	> ref -os -off: turn off os

	> pull: download packages (built upon wget)

	> ref -u <file_name>: update packages (built upon wget)

	> ref -cn <old_file_name> <new_file_name>

	> ref -d <file_name>: remove file

	> ref -n <file_name>: create file

	> ref -n -folder <folder_name>: create folder

	> auth: User authentication
		-in: login
		-out: logout
		-new: create new account
		-cp: change password
